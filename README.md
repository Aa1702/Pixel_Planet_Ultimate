Pixel Planet Ultimate is a cozy, pixel-art style geography game designed to help you learn the countries of the world. Built with HTML, CSS, JavaScript, and Electron, this app transforms your desktop into a cute, gamified learning experience. It features all 195 countries, challenging players to unscramble country names and revealing their flags upon success.
